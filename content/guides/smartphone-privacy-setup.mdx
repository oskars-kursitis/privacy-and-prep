# Smartphone Privacy Setup

**iOS**
- Settings → Privacy → Tracking: Off
- Lock screen widgets: minimal
- Use passcode + Face ID, disable unlock by USB accessories

**Android**
- App permissions: deny by default
- Enable screen lock + auto-lock
- Limit background data for noisy apps
